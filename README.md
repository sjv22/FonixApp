# Fonix zensend API

## Prerequisites

ruby 2.3.3 & Rails 5.2.1

## Gems Used

1. zensend
2. phonelib

## Install dependencies

```$ cd FonixApp```

```$ bundle install```

## Database Setup

```$ cd FonixApp```

```$ rake db:create```

```$ rake db:migrate```

## Access zensend API 

Add your zensend api key in app/helpers/authentication_helper.rb
